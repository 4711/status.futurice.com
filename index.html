Transparency brings in goodness

Check out our public status page. We published the source code so you can do that too. It's all open source, and published with MIT/BSD/GPL licenses.

Unfortunately due to nature of the site, installation is rather complicated and requires multiple steps. README file gives some instructions, but that's probably not enough.

If you encounter problems with code provided by Futurice, feel free to send email to me (olli.jarva@futurice.com), and I'll do my best to help you. I can't provide any help with general configuration issues (for example, how to configure apache2, or how to setup server in general). I recommend using superuser.com for admin issues and stackoverflow.com for programming issues.

Who are we?

This code is provided by Futurice, small software company with five offices around the Europe, and about 150 employees. You can also find us on twitter and facebook.
